# basic-chat-application

<h2>Built a basic console-based chat application in C using the Winsock2 API.</h2>

<p1>This project helped me understand:

* TCP socket programming
* Client-server architecture
* How ports are used for communication
* Winsock2 API functions like:

  * socket()
  * bind()
  * listen()
  * accept()
  * send()
  * recv()
* Multi-threaded message receiving using CreateThread()

I also worked on fixing console buffering and message overlap issues during simultaneous send/receive operations.

This was a small beginner project, but it gave me practical understanding of how network communication works internally between a client and a server.

Next goal:

* Multi-client chat support
* Dynamic usernames
* internet connectivity
  </p1>
